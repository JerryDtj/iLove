## 项目介绍

一个基于gradle的婚恋网站. 
  - 下载运行需要先集成gradle
  - gradle下载安装请参考官网 https://gradle.org/install/

## 技术栈
  - Spring Security
  - Spring Security OAuth2
  - myBatis
  - myBatis plus
  - JWT
  - Redis
  - Thymeleaf
  - Bootstrap
  
## 项目搭建

  **运行环境：**

    jdk1.8+gradle。
  
  **数据库配置：**
 
  数据库mysql
